# 📦 CoreInventory System
### Warehouse & Inventory Management Platform

CoreInventory is a **full-stack inventory management system** designed to help businesses manage products, warehouses, and stock movement efficiently.

The system includes:

📱 **Mobile Application (Android)** – for warehouse workers and managers  
🌐 **Web Dashboard** – for administrators and analytics  

Both applications communicate with a **REST API backend** to manage real-time inventory data.

---

# 🚀 System Overview

CoreInventory provides tools to:

- Track product inventory
- Manage warehouses and storage locations
- Monitor stock movement
- Analyze inventory data
- Manage suppliers and purchase orders

The system is built to be **scalable, modern, and easy to use**.

---

# 📱 Mobile Application (Android)

The **Flutter-based Android application** allows warehouse staff to manage inventory directly from mobile devices.

### Mobile Features

- Secure Login System
- Product scanning and lookup
- Inventory updates
- Stock adjustment
- Warehouse location tracking
- Dashboard overview

---

# 🌐 Web Dashboard

The **Web Admin Dashboard** provides a complete management interface.

### Web Features

- Product management
- Inventory overview
- Warehouse management
- Supplier management
- Stock analytics
- Purchase order tracking
